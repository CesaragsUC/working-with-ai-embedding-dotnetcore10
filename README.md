# working-with-ai-embedding-dotnetcore10
## Example its using .Netcore 10
## Packages envolved:
- Microsoft.Extensions.AI
- Microsoft.Extensions.AI.Abstractions
- Microsoft.Extensions.AI.OpenAI
- Microsoft.SemanticKernel
- Pgvector.EntityFrameworkCore

How to transforming complex data (text, images, audio) into numerical vectors (arrays of numbers) that capture their meaning and relationships, allowing AI models to understand, compare, and process information more efficiently, forming the foundation for tasks like semantic search, recommendations, and understanding language.
